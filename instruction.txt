1. Скачать проект и перейти в него
    git clone https://gitlab.com/denoiselab1/block
    cd block
    git checkout dev-calija
    git submodule update --init --recursive

2. Запустить докер (возможно прийдется залогиниться - "docker login")
    docker-compose up --build

3. Запустить Backend

    Зайти в папку:
        cd Backend/

    Заполнить файл .env по примеру .env.example;
    
    Создать окружение:
        python -m venv venv
            или
        python3 -m venv venv

    Зайти в окружение:
        На macOS или Linux:
            source venv/bin/activate
        На Windows (PowerShell):
            venv\Scripts\Activate.ps1
    
    Скачать зависимости:
        pip3 install -r requirements.txt
            или
        pip3 install -r requirements.txt

    Запустить сервер (только с активированным докером)
        uvicorn main:app --reload --host 0.0.0.0 --port 8000

4. Запустить Bot:

    Зайти в папку:
        cd Bot/

    Заполнить файл .env по примеру .env.example;

    Создать окружение:
        python -m venv venv
            или
        python3 -m venv venv

    Зайти в окружение:
        На macOS или Linux:
            source venv/bin/activate
        На Windows (PowerShell):
            venv\Scripts\Activate.ps1
    
    Скачать зависимости:
        pip3 install -r requirements.txt
            или
        pip3 install -r requirements.txt

    Запустить бота (только с активированным докером)
        python3 bot.py

4. Запустить Frontend:

    Зайти в папку:
        cd Web/

    Заполнить файл .env по примеру .env.example;

    Скачать зависимости:
        pnpm i
    
    Запустить приложение:
        pnpm run dev
